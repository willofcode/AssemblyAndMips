           MIPS Assembly Program: Linked List Operations
######################################################################

Author: William Ng

######################################################################

Description:
------------

This MIPS assembly language program showcases linked list operations,
including insertion, deletion, and traversal. The program defines a simple
linked list structure and provides functions for adding elements to the
list (insertion), removing elements from the list (deletion), and
traversing the list to print or manipulate its contents.

Additional note:
----------------

This linked list interactive program only accomodates integers.
selection on the menu selection follows from 0 to 5

######################################################################

Instructions:
-------------

1. Open the MIPS assembly code file (e.g., `linked_list_operations.asm`) using
   a MIPS assembly language IDE or text editor.

2. Assemble and run the program using a MIPS assembler and simulator.

3. linked list node are empty in the beginning. Insertion of 0 must be intialized
    to start adding element on the linked list.

4. Modify the linked list structure or the elements to be inserted into the
   list according to your requirements.
